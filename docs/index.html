dcode

Distributed Code Editor.

Usage

This editor is a thin wrapper around CodeMirror, primarily intended to be used with dterm. For that reason there are three installable commands available to open files from your terminal in a new dcode tab. You can install and use them like so:

> term install dat://dcode.hashbase.io/commands/code.js
> code path/to/file.txt

# With Emacs keybindings
> term install dat://dcode.hashbase.io/commands/emacs.js
> emacs path/to/file.txt

# With vim keybindings
> term install dat://dcode.hashbase.io/commands/vim.js
> vim path/to/file.txt

If the file in question does not exist, it will be created. Any changes you make will be saved automatically.
